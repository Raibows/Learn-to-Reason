## Democratizing Reasoning Ability

Code for paper "Democratizing Reasoning Ability: Tailored Learning from Large Language Model" accepted by EMNLP 2023.

1. python 3.8.16
2. cuda 11.7
3. pytorch 1.13.1
4. transformers 4.28.1
5. accelerate 0.18.0
6. datasets 2.10.1
7. deepspeed 0.9.1
